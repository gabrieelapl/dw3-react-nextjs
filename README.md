# 📚 DW3 - React com Next.js

Repositório destinado às atividades desenvolvidas em aula na disciplina de Desenvolvimento Web 3 (DW3), utilizando React com Next.js.

---

## ⚙️ Criar projeto React (Next.js)

Para iniciar um novo projeto durante as aulas, utilize:

```bash
npx create-next-app .
```

## ▶️ Rodar o projeto

Para executar o projeto localmente:

```bash
npm run dev
```
A aplicação estará disponível em:

```bash
http://localhost:3000
```
